# Alpina Recipes and Articles

Magento 2 module for managing recipes and articles.

## Installation

1. Copy module to `app/code/Alpina/RecipesAndArticles`
2. Run setup commands:
   ```bash
   bin/magento module:enable Alpina_RecipesAndArticles
   bin/magento setup:upgrade
   bin/magento setup:di:compile
   bin/magento cache:flush
   ```
